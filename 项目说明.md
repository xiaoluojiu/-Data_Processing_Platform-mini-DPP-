# 📊 数据分析平台 - 项目说明

## 🎯 项目概述

这是一个基于 **Streamlit** 开发的交互式数据分析平台，集成了数据加载、预处理、探索性分析、机器学习、聚类分析、降维和关联规则挖掘等功能。

## 📁 核心文件说明

### 主应用
- **main.py** - Streamlit主应用，整合所有功能模块

### 核心模块
- **data_loader.py** - 数据加载、质量评估、数据清洗
- **visualization.py** - 数据可视化（EDA图表）
- **ml_models.py** - 机器学习统一接口层
- **ml_visualization.py** - 机器学习相关可视化

### 算法实现（模块化）
所有算法已拆分到 `算法/` 目录下，每个算法独立文件：

#### 公共工具
- `算法/utils.py` - 算法公共工具模块（评估指标、特征重要性、数据验证等）

#### 回归算法
- `算法/回归/线性回归.py`
- `算法/回归/K近邻回归.py`
- `算法/回归/决策树回归.py`

#### 分类算法
- `算法/分类/逻辑回归.py`
- `算法/分类/朴素贝叶斯.py`
- `算法/分类/K近邻分类.py`
- `算法/分类/决策树分类.py`（支持CART/ID3/C4.5）

#### 聚类算法
- `算法/聚类/K均值聚类.py`
- `算法/聚类/DBSCAN聚类.py`

#### 其他算法
- `算法/降维/主成分分析.py`
- `算法/关联规则/Apriori关联规则.py`
- `算法/关联规则/FPGrowth关联规则.py`
- `算法/模型解释/SHAP解释器.py`
- `算法/数据准备.py`

## 🔧 开发说明

### 代码结构
- 所有算法实现模块化，便于维护和扩展
- `ml_models.py` 作为统一接口层，保持向后兼容
- UI使用Streamlit + 自定义CSS美化

### 性能优化
- 大数据集自动采样（聚类分析）
- 轮廓系数计算优化（采样后计算）
- 散点图点数限制（避免浏览器卡顿）

### 扩展性
- 新增算法：在对应目录下创建新文件，实现 `train()` 或 `fit()` 函数
- 在 `ml_models.py` 中导入并添加到 `model_map`

## 📌 注意事项

1. 确保已安装所有依赖：`pip install -r requirements.txt`
2. 使用Conda环境时，确保环境名为 `lxt`
3. 背景图片 `background.jpg` 为可选文件，不存在时使用默认背景
